# Login + Signup UI Template

Clean auth forms compatible with any provider.

## Usage

```
app/login/page.tsx   → import Login from "./login"
app/signup/page.tsx  → import Signup from "./signup"
```

## Works With

- Clerk
- NextAuth
- Auth0
- Firebase Auth
- Custom backend

## Customization

- Add social login buttons
- Add "Forgot password" link
- Connect to your auth API
